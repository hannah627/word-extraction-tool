Word Extraction Tool

Welcome!

This is a tool to help with building a custom sentiment analysis lexicon. It allows users to upload a text or texts, standardizes the words by removing capitalization and punctuation, removes basic stop words, and then goes through each token in the text(s) one at a time. For each token, the user will have two options - to add the word to a lexicon, or to ignore it. Regardless of the user's choice, the word will be removed from the rest of the text so that it's only shown to the user once.
